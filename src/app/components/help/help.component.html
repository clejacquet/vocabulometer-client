Requirements

Only Windows

Sorry, Linux or MacOS can't be used, as the website is powered by the Tobii API, only compatible with Windows.

A Tobii Eye-Tracker installed and working

In order to use this app a Tobii Eye-Tracker is required. So far the following models have been tested and work with this website:

  • Tobii EyeX
  • Tobii 4C

Installation of the drivers and calibration must have been done beforehand.

The Gaze Provider downloaded and launched

For a correct use of the website a software called "Gaze Provider" is required. It analyzes your gaze and provides a stream of data to the website in real-time.

See the link below for download.

Download

Download the provider.exe.

Then, launch it and you're done! You can go back to the Home page and start reading a text.

Browser compatibility

The following browsers have been tested and are compatible with Vocabulometer:
  • Google Chrome v65.0+
  • Mozilla Firefox v59.0+
  • Microsoft Edge v41+ (Animations not working)

Known issues

1. Windows zoom level at boot

If your computer booted on Windows with a zoom level higher than 100% (e.g. 125%, 150%...) then the reading detection will fail. However, booting at zoom level 100% and then changing to 125% or more works.

2. Browser zoom level

The browser zoom level should not be changed and stay at the default 100% value.

3. Download bar open

On some browsers (such as Google Chrome) if the download bar is open the analysis is a bit less accurate.
